🔘 STILE PULSANTI BUBBLE
═══════════════════════════════════════════════════════════════════════

CSS Applicato:
• Border-radius: 20px (arrotondati stile bubble)
• Transizione: 0.3s cubic-bezier (smooth)
• Hover: scale(1.02) + translateY(-3px)
• Active: scale(0.98) + translateY(-1px)
• Shadow: box-shadow rgba(124, 111, 255, 0.25)

Animazioni Disponibili:
• bubblePulse: 2s infinite (glow effect)
• slideIn: Entrata elementi dall'alto
