# Kazka – Smart Finance App

## File inclusi
- `app.html` — App principale (PWA)
- `index.html` — Landing page
- `manifest.json` — PWA manifest
- `sw.js` — Service Worker per offline
- `icon-192.png` / `icon-512.png` — **Da generare** (vedi sotto)

## Icone PWA
Il logo Kazka viene generato dinamicamente via canvas nel browser.
Per le icone statiche PWA (home screen), genera i PNG con:
https://kazka.app/icon-generator (o usa il canvas export dall'app)

## Deploy
Carica tutti i file in `/appfinance/` su GitHub Pages.
